Cara Menggunakan
  1. Test Connection - Cek apakah Customer API bisa diakses
  2. Get Device - Masukkan Serial Number ONU untuk mengambil data (jika sudah ada di database)
  3. Save Location - Simpan data pelanggan baru (SN, username, password, koordinat)
  4. Test Login - Test login dengan username dan password yang sudah disimpan
Tip: Jika API mengembalikan error, pastikan:
  • Database MariaDB/MySQL sudah running
  • Tabel onu_locations sudah dibuat (jalankan ./install.sh)
  • Kredensial database di customer_api.php sudah benar
